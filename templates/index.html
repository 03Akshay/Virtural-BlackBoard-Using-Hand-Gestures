VR BlackBoard Features

Hand Tracking

The project is entirely based on hand tracking, which allows users to write on the board using their hands.

Real-Time Drawing

The application provides real-time drawing capabilities, ensuring that users can see their input immediately.

FPS Display

Displaying the frame rate per second (FPS) helps in monitoring the performance of the application.

Saving Drawings

Users can save their board writings by pressing the save button, which stores the content in a local database.

Clearing the Whiteboard

The whiteboard can be easily cleared by pressing the clear button, removing all the content.

Database Integration

The project integrates with a database to store and manage all drawings and notes effectively.
